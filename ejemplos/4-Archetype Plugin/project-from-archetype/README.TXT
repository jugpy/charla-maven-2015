Creado desde un arquetipo existente en maven central:
1) Se ejecuta $mvn archetype:generate (Este comando corre en modo interactivo, puede tardar en aparecer el prompt nuevamente ya que se descargan todos los catálogos de arquetipos disponibles.
2) Se introduce el filtro "org.codehaus.mojo.archetypes:webapp-javaee6" para obtener el número del arquetipo.
3) Se introduce el número que aparece en la consola para este arquetipo.
4) Se escoje la mayor versión disponible(1.5)
5) Se introdujeron los valores para groupID, artifactID y version.
